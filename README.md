# 🎓 Student Management System

A simple command-line Student Management System built with Python to practice Object-Oriented Programming (OOP), file handling, and basic CRUD operations.

## 📌 Features

- Add a new student
- View all students
- Search for a student by name
- Delete a student
- Store student records in a text file

Student records are stored in the following format:

John|20|Computer Science
Mary|19|Accounting
David|22|Physics

---

## 🛠️ Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- File Handling (`open`, `read`, `write`, `append`)
- Class Methods
- Exception Handling
- String Methods (`split()`)

## ▶️ Running the Project

Clone the repository:

```bash
git clone https://github.com/yourusername/student-management-system.git
```

Navigate into the project folder:

```bash
cd student-management-system
```

Run the application:

```bash
python script.py
```
